# Todos-List\n\nA simple React Todo List application built with Create React App. Features include adding, viewing, editing, and deleting todos.\n\n## Features\n- Add new todos\n- Mark todos as completed\n- Edit existing todos\n- Delete todos\n- Responsive UI with Header, Footer\n\n## Demo\n![Todo App Screenshot](https://via.placeholder.com/800x400?text=Todos-List)\n\n## Getting Started\n\n### Prerequisites\n- Node.js\n\n### Installation\n1. Clone the repo:\n```sh\ngit clone https://github.com/muskanahirwar/Todos-List.git\ncd Todos-List\n```\n2. Install dependencies:\n```sh\nnpm install\n```\n\n### Run the app\n```sh\nnpm start\n```\nOpen [http://localhost:3000](http://localhost:3000)\n\n### Build for production\n```sh\nnpm run build\n```\n\n## Tech Stack\n- React\n- CSS Modules\n\n## License\nMIT\n
